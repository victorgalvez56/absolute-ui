# Absolute UI Playground

Liquid-glass React Native design system, rendered here through
**react-native-web** so you can explore every primitive without
booting a simulator.

## Personalities

Use the **Personality** toolbar at the top to swap the active
theme — Aurora, Obsidian, Frost, or Sunset. Every story reads
the selection from a shared `AbsoluteUIContext`, so the change
propagates through glass recipes, motion identity, and semantic
colors.

## Controls

Every primitive story exposes interactive knobs in the
**Controls** tab — tweak `elevation`, `radius`, `disabled`,
`invalid`, and more to preview permutations live. The
**AllPersonalities** stories render the same component across
all four personalities side-by-side for quick visual QA.

## Screens

The `Screens/` section composes the primitives into full
end-to-end flows (Login, Settings, Now Playing) framed inside a
390×844 phone chrome so the result reads like a real device
instead of a web card.
