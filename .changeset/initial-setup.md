First public release. Ships the liquid-glass primitives (`GlassSurface`,
`GlassButton`, `GlassCard`, `GlassSheet`, `GlassNavBar`, `GlassTabBar`,
`GlassModal`, `GlassToast`, `GlassInput`, `GlassSwitch`, `GlassSlider`,
`GlassPicker`), the four personality themes (Aurora, Frost, Obsidian,
Sunset), APCA contrast utilities, preference resolvers, and the shared
MSW mock server. Each package exposes CJS + ESM builds with matching
TypeScript declarations.
